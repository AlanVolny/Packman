Name:
  Nun
Version:
  1.1
Description:
  Nun sprites created as drop-in replacements for Stendhal. Svetlana
  Kushnariova's sprite template is used for the base. The outfit is
  re-worked from Charles Gabriels nun.

  The file nun_orig-black.png is simply a scaled & touched up version
  of the original sprite by Charles Gabriel (Antifarea). It does not
  use the Cabbit template as a base.
Details:
  - Orientation: orthogonal
    - N/E/S/W format for Stendhal.
    - S/W/E/N format for RPG Maker & RPG Boss.
  - Dimensions: 24x32 & 48x64 (scaled)
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
